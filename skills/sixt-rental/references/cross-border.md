# Cross-Border Travel Restrictions

Vehicle category determines which countries you can enter. Countries are grouped into four zones.

## Zones

| Zone | Countries |
|------|-----------|
| **I** | Andorra, Austria, Belgium, Denmark, Finland, France, Germany, Gibraltar, Great Britain, Ireland, Italy, Liechtenstein, Luxembourg, Monaco, Netherlands, Norway, Portugal, San Marino, Sweden, Switzerland, Spain, Vatican |
| **II** | Croatia, Czech Republic, Poland, Slovakia, Slovenia |
| **III** | Estonia, Hungary, Latvia, Lithuania |
| **IV** | All countries not in Zone I, II, or III — **entry not permitted** |

## Restrictions by ACRISS category

| Allowed zones | ACRISS pattern | Description |
|---------------|----------------|-------------|
| **I only** | `W***`, `X***` | Luxury Elite and Special class |
| **I only** | `GVAR`, `UVAR` | Premium Elite vans |
| **I only** | `W*S*`, `X*S*` | Special Cars from Luxury Elite upward |
| **I only** | *(all Sports & Luxury Cars)* | Regardless of ACRISS code |
| **I + II** | `P***`, `U***`, `L***` | Premium, Premium Elite, Luxury |
| **I + II** | up to `U*S*` | Special Cars up to Premium Elite |
| **I + II** | *(all vans and trucks)* | Regardless of ACRISS code |
| **I + II + III** | *(all other categories)* | Economy, Compact, Intermediate, Standard, etc. |

To check a specific offer: match the first character of the ACRISS code against the patterns above. Most common rental categories (C, D, I, S, F) are Zone I+II+III — only premium/luxury categories (P, U, L, W, X) and vans/trucks have tighter restrictions.

## Additional rules

- **Leaving the European continent is not permitted.** This includes the Spanish exclaves Ceuta and Melilla.
- **Ferry transport** is allowed within the zone restrictions above. Damage during ferry crossings is covered by the protection booked in the rental contract.
- **Violating these rules voids all protection** except mandatory liability insurance. The renter is personally liable for any damage.
- The renter must comply with all official entry, exit, and access requirements of each country.
